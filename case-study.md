## 1. 📌 Описание проекта

### Бизнес-задача
Автоматизировать обработку текстовых обращений клиентов. Модель определяет:
1. **Настроение** (сентимент): позитив / негатив / нейтраль
2. **Тему обращения**: жалоба, вопрос, заказ, возврат и т.д.

### Цели
- Обучить модель сентимент-анализа с accuracy > 90%
- Обучить мультиклассовую модель (10+ категорий) с accuracy > 85%
- Создать pipeline для предсказания на новых текстах
- Задокументировать процесс и результаты

### Предобработка данных ✅
**Дата:** 26.06.2026  
**Токенизатор:** DistilBERT (distilbert-base-uncased)  
**Длина последовательности:** 512 токенов  
**Метод:** Padding + Truncation

**Что сделано:**
- Текст разбит на токены (слова/части слов)
- Каждому токену присвоен числовой ID
- Добавлены attention masks (чтобы модель знала, где реальный текст, а где padding)
