## 1. 📌 Описание проекта

### Бизнес-задача
Автоматизировать создание маркетингового контента с помощью локального AI.

### Цели
- Создать workflow в n8n для генерации постов
- Интегрировать с Ollama (Llama3) для работы без интернета
- Создать Telegram-бота для удобства использования

---

## 2. 🏗️ Техническое описание

### Инструменты
- **n8n** — платформа автоматизации workflows
- **Ollama** — локальный AI-сервер (Llama3)
- **HTTP Request Node** — взаимодействие с API Ollama
- **Telegram Bot API** — интерфейс для пользователей

### Архитектура
Триггер → Edit Fields (topic) → HTTP Request (Ollama) → Вывод результата
