### 2. 🎬 AI Video Creator (GUI)
**[project-04-ai-video-creator/](project-04-ai-video-creator/README.md)**

Автоматическое создание видео с AI-озвучкой из стоковых материалов.

**Возможности:**
- 🎥 Автоматический поиск видео (Pexels API)
- 🗣 AI-озвучка текста (Microsoft Edge TTS)
- 🎬 Автоматический монтаж
- 🎨 Красивый GUI интерфейс
- 🌍 Поддержка русского и английского

**Технологии:**
- Pexels API (стоковые видео)
- Edge TTS (синтез речи)
- MoviePy (монтаж видео)
- CustomTkinter (GUI)

**Результат:** Видео 2.9 MB за несколько минут

### 3. 📊 Excel Report Generator
**[project-06-excel-report-generator/](project-06-excel-report-generator/README.md)**

Профессиональный инструмент для анализа данных и генерации отчётов.

**Возможности:**
-  Расширенная статистика (сумма, среднее, медиана, процентили)
- 📊 5 типов графиков (динамика, топ, категории, города, распределение)
- 💾 Экспорт в Excel с 5 листами
- 🎨 GUI на CustomTkinter (тёмная тема)
-  Поддержка CSV и Excel

**Технологии:**
- Pandas (анализ данных)
- Matplotlib + Seaborn (визуализация)
- OpenPyXL (Excel форматирование)
- CustomTkinter (GUI)

**Результат:** Автоматические отчёты за секунды
